#### ✨ 项目亮点：
- **傻瓜式安装**：无需提前安装 Node.js、Git 或 NPM，双击 EXE 即可完成。
- **全技能集成**：默认安装全部技能（Skills），无需额外手动扩展。
- **内置优化**：默认配置阿里源，确保国内环境下运行稳定。
- **干净卸载**：程序目录下附带一键卸载 EXE，不残留系统垃圾。

#### 常见问题：
- **缺少VCRUNTIME140.dll**：
如果您收到了“由于您的电脑缺失 VCRUNTIME140.dll 文件，该程序无法启动。请尝试重新安装程序以修复此问题。”或者“由于未找到 VCOMP110.DLL，代码无法继续执行。重新安装程序可能会修复此问题。”的错误提示，这代表您需要重新在 PC 上安装 Microsoft Visual C++ Redistributable。您可以从 Microsoft 的下载页面[https://www.microsoft.com/zh-cn/download/details.aspx?id=48145](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)获取并安装该运行时组件。
